# AgroAssist

STEPS TO RUN THIS APP :-

1: Install all the files present in this project.
2: Download required libraries by running the command in terminal "pip install -r requirements.txt" .
3: Install Xampp Server -> Start MySQL & Apache server.
4: Inside "main.py" file, Change 17th line of code as " app.config['SQLALCHEMY_DATABASE_URI'] = 'mysql://root:@localhost:<port_no>/<database_name>' ".
5: In terminal run "python main.py"
6: Click on the url to open the app.
